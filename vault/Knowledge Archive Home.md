# Knowledge Archive Home

> A living intelligence system. Every note is a node. Every connection is a pathway to insight.

---

## Dual-Layer Architecture

| Layer | Location | Purpose | For Whom |
|-------|----------|---------|----------|
| **Human Layer** | `00 Inbox/` through `13 Meta Analysis/` | Workflow portals, navigation anchors, daily use | The operator |
| **Machine Layer** | `wiki/`, `sources/`, `schema/` | AI-maintained knowledge graph, immutable sources, governance | Agents |

The Human Layer preserves a numbered workflow (00–13). The Machine Layer adds a Karpathy-inspired three-tier stack underneath: `sources/` for raw material, `wiki/` for the knowledge graph, `schema/` for governance.

> [!tip] **New to the system?** Start with the [[README]] for the full architecture, or jump to a workflow: [[workflows/inbox-triage|inbox-triage]], [[workflows/daily-rhythm|daily-rhythm]], [[workflows/forward-references|forward-references]], [[workflows/maintenance-protocol|maintenance-protocol]].

## How to Use This Archive

### When You Feel Lost
1. Read [[wiki/__graph_index__|Graph Index]] to understand the topology
2. Review [[01 Identity|01 Identity]]
3. Check [[13 Meta Analysis]] for system health

### When You Need to Decide
1. Consult [[02 Principles|02 Principles]]
2. Search [[06 Decision Archive|06 Decision Archive]] for similar situations
3. Apply a framework from [[05 Framework Library|05 Framework Library]]

### When You Need to Create
1. Mine [[wiki/synthesis/00-synthesis-engine-hub|Synthesis Hub]] for cross-domain material
2. Review [[10 Ideas & Opportunities|10 Ideas & Opportunities]]
3. Follow a [[templates/project-note|project template]]

### When You Need to Execute
1. Follow the relevant SOP in [[07 Systems & SOPs|07 Systems & SOPs]]
2. Use a prompt from [[08 Prompt Library|08 Prompt Library]]
3. Track progress in [[11 Timeline|11 Timeline]]

## Maintenance Protocol

- **Daily:** Five-minute evening inbox pass — see [[workflows/daily-rhythm|daily-rhythm.md]].
- **Weekly:** Process [[00 Inbox]], review [[06 Decision Archive|06 Decision Archive]].
- **Monthly:** Review [[13 Meta Analysis]] for quality drift, run a health check.
- **Quarterly:** Deep read of [[wiki/synthesis/00-synthesis-engine-hub|Synthesis Hub]], reconcile the wiki.
- **Annually:** Full audit, eliminate duplicates, update the evolution map.

See [[workflows/maintenance-protocol|maintenance-protocol.md]] for the complete schedule.
